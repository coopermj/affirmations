URL & Routing Structure

How visitors and editors navigate the app

Public Routes
/[slug] Direct page — public or checks for ?t=token if private
/c/[slug] Random published page from that category
/random Random published page from entire site
/ Redirects to /random
Private pages require ?t=[uuid] — without the token they return a 404, not a 403, so their existence isn't revealed.
Admin Routes (auth-protected)
/admin Dashboard overview
/admin/pages Page list — create, edit, delete, reorder
/admin/pages/[id] WYSIWYG editor + settings panel + preview
/admin/backgrounds Upload & manage background media
/admin/fonts Google / uploaded / Adobe font management
/admin/categories Create & organize categories
/admin/users Admin only — create & manage editor accounts
NFC Tag → Page Flow
NFC tag URL → /[slug] or /c/[slug] or /random → Full-screen affirmation page
